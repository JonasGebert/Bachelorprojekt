## Projektziel (Kurzfassung)

Entwicklung, Erprobung und Dokumentation eines Laborversuchs (2 Laborviertel, 3. Semester), in dem Studierende ein reales Metallbauteil mit mindestens einem Kern im Handformverfahren (Ober-/Unterkasten) sicher gießen. Eigenleistung: Bauteilkonstruktion, Modell, Kernkasten, Formkästen, Materialauswahl über Vorversuche, Versuchsskript.
